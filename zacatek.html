Tvoříme webové stránky za pomocí jazyků HTML a CSS


HTML

Neboli Hypertext Markup language je jazyk používaný pro tvorbu webových stránek, které jsou propojeny hypertextovými odkazy. HTML je hlavním z jazyků pro vytváření stránek v systému World Wide Web, který umožňuje publikaci dokumentů na Internetu.

CSS

Neboli kaskádové styly jsou pro popis způsobu zobrazení elementů na stránkách napsaných v jazycích HTML, XHTML nebo XML. Jazyk byl navržen standardizační organizací W3C a autorem návrhu byl Håkon Wium Lie.
